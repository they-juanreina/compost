# plugin

Claude Code plugin packaging: slash commands (`/compost-ingest`, `/compost-status`, `/compost-tag`, `/compost-chat`), wrapped skills (refactored `thematic-coding`, `journey-mapping`, `querying-research-knowledge`, `saturation-analysis`), and an MCP server exposing compost operations as typed tools. All three call the same Node core in `cli/`.

Cowork-registry packaging ships in M3 from the same source.

See [ROADMAP.md § Agent integration](../ROADMAP.md#agent-integration).
