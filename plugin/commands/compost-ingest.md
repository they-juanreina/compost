---
description: Ingest a file or folder into a compost seed
allowed-tools: Bash(compost ingest:*)
---

Ingest the path the user gives into the seed's job queue: `compost ingest <path>`. Audio/video route to the transcriber; PDF/DOCX/PPTX/CSV/MD route to legacy-ingest. Report how many files were queued vs skipped, and any unsupported files.

Arguments: $ARGUMENTS
