# compost

**Local-first, AI-first research analysis harness for coding agents and humans.**

Ingest interview audio and video. Transcribe with descriptive cues — silences, laughter, sighs, prosody hints — paired with screenshots captured at salient moments. Highlight, comment, and tag across sessions until the coded material is ready to harvest into synthesis. Nothing wasted — every session breaks down into something the rest of your research can feed on.

Compost runs on your machine. The filesystem is canonical. Provenance is git-style — every highlight, code, theme, and insight carries a three-actor lineage (researcher / agent / AI-suggestion). The same operations are reachable from a CLI (any coding agent can call it), a Claude Code plugin (slash commands + MCP), and a local Next.js web UI for the work that needs visual affordances.

Compost composes [research-os](https://github.com/they-juanreina/research-os) skills as tools and supersedes its router with autonomous loops.
